隐私政策

最后更新：2026年4月22日

概述

ClaudeTokenMonitor（以下简称"本应用"）是一款用于监控 Claude API Token 用量与成本的 macOS 菜单栏应用。我们致力于保护您的隐私。

数据收集

本应用不收集任何数据。本应用不会收集、传输、存储或共享任何个人信息或使用数据。

工作原理

本应用仅读取存储在您设备本地 ~/.claude/projects/ 目录下的使用数据。所有数据处理均在您的设备上本地完成，任何数据都不会发送至任何服务器、第三方或外部服务。

无网络通信

本应用完全离线运行，不发起任何网络请求，不与任何外部服务器通信。

第三方服务

本应用不使用任何第三方分析、广告或追踪服务。

政策变更

如果我们更新本隐私政策，将会更新上方的"最后更新"日期。

联系我们

如果您对本隐私政策有任何疑问，请在我们的 GitHub 仓库提交 Issue。
